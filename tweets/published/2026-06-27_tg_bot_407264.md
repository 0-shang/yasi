GitHub 上有个项目叫 ai-berkshire

不是 AI 公司，而是用 Claude Code 搭了一套价值投资研究框架。

他们把巴菲特、芒格、段永平、李录这四个人的方法论，写成了多 Agent 的协作流程，让 AI 互相辩论、交叉验证。

听起来像用 AI 复制一群大师的脑回路，然后对着财报和公司做分析。

目前还是个实验性项目，但思路挺有意思的。
---
对我来说，最有价值的部分不是“AI 荐股”，而是把大师们的思维模型显式化、可操作化。

当你写一个 Agent 时要先问：芒格会先看什么？段永平会算哪个指标？

这本身就是一个很好的学习工具。
