Cognee 是个挺有意思的开源项目，专为 AI Agent 做长期记忆。

它本质上是一个自托管的知识图谱引擎，能让你的 Agent 跨会话记住东西。

这就意味着你不需要每次对话都从头开始，Agent 能记住之前聊过什么。

GitHub 上已经有 2.4 万星了，Python 写的，可以自己部署。

对于做 AI 应用的人来说，这个方向确实挺刚需的。
