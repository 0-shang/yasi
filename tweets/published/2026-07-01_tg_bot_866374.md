GitHub 上有个挺有意思的项目：ai-berkshire。

它把巴菲特、芒格、段永平、李录四位的价值投资方法论，封装成了一套基于 Claude Code / Codex 的研究框架。

核心思路是让多个 AI Agent 并行工作，从不同角度分析一家公司 —— 一个看护城河，另一个算安全边际，第三个评估管理层。

最后汇总成类似伯克希尔内部讨论纪要的东西。

虽然离真正的“AI 巴菲特”还差很远，但这个方向很聪明：用多 Agent 模拟多元思维模型，而不是让一个模型瞎猜。
