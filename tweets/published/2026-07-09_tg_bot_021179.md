GitHub 上有个叫 'last30days' 的开源项目，几天涨了五万多星。

说白了就是一个 AI agent skill：你给它一个话题，它就自己去 Reddit、X、YouTube、HN、Polymarket 全网搜，最后合成一份有根有据的摘要。

相当于你雇了个实习生，翻遍过去30天全网关于某个话题的讨论，然后回来给你口述重点。

Python 写的，开源可改。

有需求直接 GitHub 搜 last30days。
